# 社区行为准则

本项目希望保持一个直接、专业、可协作的社区环境。

## 我们鼓励的行为

- 基于事实讨论问题
- 尊重不同经验水平的参与者
- 在反馈中给出可执行建议
- 对设计、实现、测试和文档保持建设性态度
- 聚焦问题本身，而不是针对个人

## 不可接受的行为

- 人身攻击、羞辱、讽刺或恶意挑衅
- 歧视、骚扰、威胁或仇恨言论
- 恶意刷屏、故意制造争吵或破坏讨论
- 未经允许公开他人隐私、密钥、凭证或敏感信息
- 明知会破坏项目秩序却持续重复相同行为

## 适用范围

本准则适用于以下场景：

- GitHub issue
- Pull Request
- 代码评审
- 讨论区
- 项目相关即时沟通场景

## 执行原则

维护者可以根据实际情况采取以下措施：

- 提醒与引导
- 编辑或隐藏不当内容
- 锁定讨论
- 拒绝合并贡献
- 暂时或永久限制参与

执行时会优先考虑社区秩序、协作效率和参与者安全。

## 如何反馈

如果你认为某些行为违反了本准则，请通过仓库维护者提供的私有联系方式反馈，并尽量附上上下文。

## 结论

欢迎严格讨论技术问题，但不欢迎破坏协作环境的表达方式。

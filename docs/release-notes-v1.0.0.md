# v1.0.0 发布说明

## 版本定位

`v1.0.0` 是项目首个可公开发布版本，目标是提供一套可继续开发、可联调、可部署、可开源协作的现代若依后台工程。

本版本同时包含：

- Vue 3 前端管理后台
- NestJS + Prisma 配套后端
- 设计、开发、部署、发布所需的完整文档链路

## 本次发布亮点

### 前端

- 使用 `shadcn-vue + Tailwind CSS v4` 重构若依后台视觉与交互
- 接入动态菜单、权限、标签页、主题系统、布局设置
- 核心 CRUD 页已改成真实业务页，不再依赖统一配置渲染
- 表格、查询区、弹窗、操作栏、日期选择、富文本、头像裁剪均已抽成共享组件
- 已完成移动端适配、顶部标签滚动、隐藏式滚动条与弹窗滚动优化

### 后端

- 使用 `NestJS + Prisma` 对齐若依接口契约
- 已覆盖认证、系统管理、监控、代码生成、上传、导出、Swagger、Druid 兼容页
- 支持 `Prisma 优先 + mock 回退`，方便数据库未就绪时继续联调
- 导出已接成真实文件流，上传已接成真实落盘链路

### 工程与交付

- 提供数据库初始化、种子数据、Docker、PM2、Nginx 部署方案
- 提供 CI、README、贡献指南、Issue / PR 模板、安全策略和行为准则
- 提供设计、维护、部署、发布全链路文档

## 已知边界

- `tool/build` 是增强版表单设计工作台，不是原版若依 1:1 实现
- `monitor/druid` 在 NestJS 版本里是诊断页兼容实现，不是原版 Druid 控制台
- 当前后端更适合联调和二开，生产落地前仍建议结合真实数据库初始化、部署环境和监控策略做最终加固

## 建议的发布摘要

一套基于 `Vue 3 + shadcn-vue + Tailwind CSS v4` 的现代若依后台，同时包含 `NestJS + Prisma` 配套后端。支持动态菜单、权限、主题系统、监控、代码生成、上传导出与部署初始化链路，可直接作为联调、二开和开源协作基线。
